2. 새 브랜치 생성 (`git checkout -b feature/add-new-api`)
3. 변경사항 커밋 (`git commit -am 'Add new API'`)
4. 브랜치에 Push (`git push origin feature/add-new-api`)
5. Pull Request 생성
